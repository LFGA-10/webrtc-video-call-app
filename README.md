# VisionLink WebRTC 🎥

A professional, low-latency video conferencing interface built with a focus on immersive user experience and secure communication.

![Video Call Preview](https://images.unsplash.com/photo-1616587226960-4a03badbe8bf?auto=format&fit=crop&q=80&w=1000)

## ✨ Features
- **Immersive Video Grid**: Optimized for high-fidelity participant visualization.
- **Dynamic Controls**: Intuitive management of mic, video, and active calls.
- **Real-time Chat**: Integrated sidebar for seamless in-call messaging.
- **Premium Aesthetics**: Dark-mode optimized with glassmorphic elements.
- **Fluid UI**: Smooth transitions powered by Framer Motion.

## 🚀 Built With
- **React 18**
- **Tailwind CSS**
- **Framer Motion**
- **Lucide React**

## 🛠️ Usage
```bash
npm install
npm run dev
```
